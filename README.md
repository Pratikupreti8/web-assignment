**University Campus Portal**
Project Description

The University Campus Portal is a front-end academic web project developed using HTML5 and CSS3. The website simulates a modern university portal featuring a student resume, online registration form, campus location map, and essential institutional information.
This project focuses on responsive design, clean UI layout, and proper use of web development standards, making it suitable for academic evaluation and learning purposes.

**Project Objectives**

To build a fully responsive university website interface

To apply semantic HTML5 elements for structured content

To implement modern CSS concepts including Flexbox, Grid, and CSS variables

To design a student resume and registration system UI

To embed external services such as Font Awesome icons and Google Maps

**Technologies & Tools Used**

HTML5 – Structuring web content

CSS3 – Styling, layout, and responsiveness

Font Awesome – Icons for UI and social links

Google Maps Embed API – Displaying campus location
